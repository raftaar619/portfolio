AI-Powered Document Retrieval

This project utilizes a chatbot system that leverages Retrieval-Augmented Generation (RAG) to answer queries by retrieving relevant information from documents. Using deep learning techniques for text extraction and embedding, the system ensures accurate and context-based responses.

The project was deployed using Flask and AWS services like EC2 and S3 to handle real-time document processing, offering both scalability and performance optimization.
